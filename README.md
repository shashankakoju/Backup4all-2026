## ❓ FAQ

**Is there any risk of my data being corrupted or lost during backup?**
No, Backup4all 2026 uses transaction-based backup logic. If a backup is interrupted, the data is rolled back to a consistent state, preventing corruption. However, we always recommend testing a restore on a sample file.

**Why does the installer ask for Administrator permissions?**
Administrator rights are required to install system-level components for scheduled tasks and to access certain protected system folders (e.g., Users, ProgramData) for full backups. The application itself runs with standard user rights once configured.

**How do I schedule a backup to run automatically?**
After installation, open the application and go to the "Scheduler" tab. You can set a backup to run daily, weekly, or at custom intervals. The program will run in the background without interfering with your work.
